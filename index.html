JavaScript ES6+ Features

Modern JavaScript Concepts with Interactive Examples

✓ let and const

let name = "Alex"; const age = 25; // age = 30; // ❌ Error: const can't be reassigned
// Output will appear here
let allows variable reassignment, while const creates a constant that cannot be reassigned.

✓ Arrow Functions

const greet = name => `Hello, ${name}`; console.log(greet("Alex"));
// Output will appear here
Arrow functions provide a concise syntax and do not have their own this context.

✓ Template Literals

const name = "Alex"; const age = 25; const msg = `Hi ${name}, you are ${age} years old.`; console.log(msg);
// Output will appear here
Template literals allow embedded expressions and multi-line strings using backticks (`).

✓ Default Parameters

function welcome(user = "Guest") { return `Welcome, ${user}`; } console.log(welcome()); console.log(welcome("Alex"));
// Output will appear here
Default parameters allow function parameters to have default values if no argument is provided.

✓ Destructuring

// Object Destructuring const user = { name: "Alex", age: 25 }; const { name: username, age: userAge } = user; // Array Destructuring const arr = [10, 20, 30]; const [first, , third] = arr; console.log(username, userAge); console.log(first, third);
// Output will appear here
Destructuring allows extracting values from objects and arrays into distinct variables.

✓ Spread and Rest

// Spread operator const nums1 = [1, 2]; const nums2 = [...nums1, 3, 4]; // Rest parameters function sum(...numbers) { return numbers.reduce((acc, n) => acc + n, 0); } console.log(nums2); console.log(sum(1, 2, 3, 4));
// Output will appear here
The spread operator expands elements, while rest parameters collect multiple elements into an array.

✓ Object Literal Enhancements

const name = "Alex"; const job = "Developer"; const person = { name, job, sayHi() { return `Hi, I'm ${this.name} and I'm a ${this.job}`; } }; console.log(person.sayHi());
// Output will appear here
Object literal enhancements provide a shorter syntax for defining properties and methods in objects.

✓ Promises

const fetchData = () => new Promise((resolve) => { setTimeout(() => resolve("Data loaded"), 1000); }); fetchData().then(result => { console.log(result); });
// Output will appear here
Promises represent the eventual completion (or failure) of an asynchronous operation and its resulting value.

✓ Async/Await

async function asyncExample() { const fetchData = () => new Promise((resolve) => { setTimeout(() => resolve("Data loaded"), 1000); }); const result = await fetchData(); console.log(result); } asyncExample();
// Output will appear here
Async/await provides a cleaner way to work with asynchronous code, making it look synchronous.

✓ Classes

class Animal { constructor(type) { this.type = type; } speak() { return `I am a ${this.type}`; } } const dog = new Animal("Dog"); console.log(dog.speak());
// Output will appear here
Classes provide a template for creating objects with shared properties and methods.

✓ Optional Chaining

const profile = { user: { name: "Alex" } }; console.log(profile.user?.name); // Alex console.log(profile.contact?.email); // undefined
// Output will appear here
Optional chaining (?.) allows reading properties deep in a chain without having to check if each reference is valid.

✓ Nullish Coalescing

let usernameInput = null; console.log(usernameInput ?? "DefaultUser"); let emptyString = ""; console.log(emptyString ?? "DefaultString");
// Output will appear here
The nullish coalescing operator (??) returns the right-hand operand when the left-hand operand is null or undefined.

✓ for...of Loop

const scores = [90, 80, 100]; let output = ""; for (let score of scores) { output += score + " "; } console.log(output);
// Output will appear here
The for...of loop creates a loop iterating over iterable objects (arrays, strings, etc.).

✓ Map and Set

// Map const map = new Map(); map.set("a", 1); map.set("b", 2); // Set const set = new Set([1, 2, 2, 3]); console.log("Map:", [...map]); console.log("Set:", [...set]);
// Output will appear here
Map is a collection of keyed values, Set is a collection of unique values.

✓ Array Methods

const arr = [1, 2, 3, 4, 5]; console.log("Includes 2:", arr.includes(2)); console.log("Find > 3:", arr.find(n => n > 3)); console.log("FindIndex of 3:", arr.findIndex(n => n === 3));
// Output will appear here
Modern array methods like includes(), find(), and findIndex() provide powerful ways to work with arrays.

✓ Object Methods

const info = { a: 1, b: 2 }; console.log("Entries:", Object.entries(info)); console.log("Values:", Object.values(info)); console.log("FromEntries:", Object.fromEntries([["x", 10], ["y", 20]]));
// Output will appear here
Object.entries(), Object.values(), and Object.fromEntries() provide ways to convert between objects and arrays.

✓ Logical Assignment

let userName = null; userName ||= "Guest"; console.log("Username:", userName); let count = 10; count &&= 20; console.log("Count:", count);
// Output will appear here
Logical assignment operators (||=, &&=, ??=) combine logical operations with assignment.

✓ Private Class Fields

class Secret { #code = 123; getCode() { return this.#code; } } const s = new Secret(); console.log("Secret code:", s.getCode());
// Output will appear here
Private class fields (prefixed with #) are accessible only within the class they are declared in.
